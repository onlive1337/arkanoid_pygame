# PyGame Arkanoid

## Требования к окружению

Для работы над кодом игры вам понадобится:

1. Git
1. Python3 для вашей системы
1. Python VENV (Устанавливается вместе с Python)

Для проверки выполните простые команды, которые должны показать текущие версии установленных программ. Если при выполнении команд выдаётся ошибка, значит прог8рамма не установлена или установлена неправильно. Пожалуйста установите или переустановите нужную программу

1. `git -v` - для проверки GIT.
1. `python3 --version` - для проверки версии Python3. Рекомендуется 3.11+

## Установка и начало работы

1. Склонируйте этот репозиторий: `git clone https://github.com/uptothetop/arkanoid_pygame.git`
1. Перейдите в папку с кодом: `cd arkanoid_pygame`
1. Настройте виртуальное окружение`python3 -m venv env`
1. Активируйте виртуальное окружение: 
    1. Для Linux, MacOSX или UNIX-систем: `source env/bin/activate`
    1. Для Windows-систем используйте Powershell: `.\venv\bin\Activate.ps1`
1. После активации в командной строке должно появиться название виртуального окружения в скобках. Например `(env)`
1. Установите зависимости: `pip install -r requirements.txt`
1. Запустите игру: `python3 main.py`

После этого должно появиться окно PyGame с заголовком "Arkanoid" - значит программа запущена верно
